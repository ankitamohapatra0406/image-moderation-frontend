# Image Moderation Frontend 

A minimal Django-based frontend that allows users to upload images and sends them to a FastAPI-based ML image moderation backend.

## Features
- Image upload UI
- Connects to ML moderation API
- Displays moderation results
- Clean and minimal design

## Tech Stack
- Django 
- HTML/CSS
- Python Requests
- FastAPI (backend)

## How it works
1. User uploads image
2. Django sends image to ML API
3. Receives moderation result
4. Displays output
